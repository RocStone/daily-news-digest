# Daily News Digest

每日 AI/科技新闻的 7 维度深度解读，由 AI 自动生成。

## 内容

- `news/` — 每日新闻解读（Hacker News + 全网信息）
- `深度解读/` — 单篇文档/技术的深度分析

每条新闻包含：核心事实、深度分析、社区讨论、批判性反思、一句话本质等维度，目标是读完解读后不需要再看原文。

## 想自己生成？

安装 [news-digest-skill](https://github.com/RocStone/news-digest-skill)，在 Claude Code 中运行 `/myhn` 或 `/news` 即可。

```bash
npx skills add RocStone/news-digest-skill
```
